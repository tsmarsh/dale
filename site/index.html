Features

🤖

Telegram Bot

Automatic bot integration handles /start, /status, /help, and /cancel. Subscribers get access, everyone else gets a payment link.

💳

Stripe Billing

Recurring subscriptions via Stripe Checkout. Webhook-driven status sync keeps everything in lockstep.

📊

Creator Dashboard

React SPA with Cognito auth. Manage rooms, view subscribers, and configure your tenant from the browser.

🏢

Multi-Tenant

One deployment serves many creators. Tenant isolation via DynamoDB key prefixes and per-tenant SSM secrets.

How it Works

1

Connect Services

Link your Telegram bot and Stripe account. Store secrets in SSM, deploy with CDK.

2

Create Rooms

Set up subscription rooms in the dashboard. Each room gets its own payment link and access controls.

3

Subscribers Pay & Get Access

Users hit /start, pay via Stripe, and instantly get access. Billing events keep status in sync automatically.
